Assembler - Detail Portofolio

Deskripsi Proyek

Proyek ini merupakan simulasi pemrograman PLC untuk proses assembling dengan menggunakan perangkat lunak TIA Portal. Dalam proyek ini, Saya memanfaatkan Factory IO sebagai platform simulasi fisik untuk mensimulasikan proses assembling secara realistis. Dengan PLCSim Siemens, Saya dapat mensimulasikan dan menguji program ladder logic yang telah dikembangkan sebelum diimplementasikan pada perangkat keras PLC yang sebenarnya.

Tujuan

Tujuan utama dari proyek ini adalah untuk mengotomatiskan proses assembling dengan memanfaatkan teknologi otomasi industri. Proyek ini bertujuan untuk meningkatkan efisiensi produksi dan mengurangi kemungkinan kesalahan manusia selama proses perakitan.

Detail Teknis Proyek: Assembler

Software yang Digunakan:

  • TIA Portal (Siemens)
  • Factory IO
  • PLCSim Siemens

Teknologi:

PLC Siemens S7-1200

Proses:

  • Sistem ini dirancang untuk mengontrol aliran material dalam proses assembling.
  • Pengoperasian conveyor untuk mengantarkan komponen ke stasiun assembling.
  • Menggunakan sensor untuk mendeteksi posisi komponen dan mengaktifkan silinder pneumatik untuk merakit produk.

Bahasa Pemrograman:

Ladder Logic

Fitur Utama:

  • Simulasi interaktif yang memungkinkan pengguna untuk melihat hasil secara real-time.
  • Integrasi antara Factory IO dan PLCSim untuk menguji dan memvalidasi logika pemrograman sebelum penerapan nyata.

Keuntungan

Proyek ini tidak hanya memberikan pemahaman mendalam tentang penggunaan PLC dan perangkat lunak otomasi, tetapi juga memungkinkan pengguna untuk berlatih merancang dan mengimplementasikan sistem otomasi dalam lingkungan yang terkendali, mempersiapkan mereka untuk tantangan di dunia industri nyata.
